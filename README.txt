STONITH Pacemaker plugin for SoftLayer API
==========================================

This is a simple plugin for the Pacemaker cluster manager to use the SoftLayer API to reboot your dedicated server or cloud computing instance.

Contributions are welcome.

Original author: Brad Jones <brad@jones.name>

Requirements
------------

* This script requires the SoftLayer API PHP implementation available [here](http://github.com/softlayer/softlayer-api-php-client).
* Extract the API package into a SoftLayer/ directory below this script; it is excluded under the .gitignore on this project.
